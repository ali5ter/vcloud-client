POST /admin/extension/vimServer/{id}/importMedia

Operation:
POST /admin/extension/vimServer/{id}/importMedia
Description:
Import media from vSphere Supported media types are ISO and FLOPPY The returned media entity will contain a task of importing process. User must periodically check this task and start using the media when the task is finished.
Since:
1.5
Input parameters
Consume media type(s):
application/vnd.vmware.admin.importMediaParams+xml
Input type:
ImportMediaParamsType
Output parameters
MediaType

Produce media type(s):
application/vnd.vmware.vcloud.media+xml
Output type:
MediaType
