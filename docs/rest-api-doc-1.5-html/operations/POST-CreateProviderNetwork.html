POST /admin/extension/externalnets

Operation:
POST /admin/extension/externalnets
Description:
Creates an external network. The name of the network must not be the same as existing one or the creation will fail. Also should be used a vim moref for valid network or dvportgroup which are not used in other external networks.
Since:
1.0
Input parameters
Consume media type(s):
application/vnd.vmware.admin.vmwexternalnet+xml
Input type:
VMWExternalNetworkType
Output parameters
VMWExternalNetworkType

Produce media type(s):
application/vnd.vmware.admin.vmwexternalnet+xml
Output type:
VMWExternalNetworkType
