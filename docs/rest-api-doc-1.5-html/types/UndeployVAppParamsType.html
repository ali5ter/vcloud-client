UndeployPowerAction string 0..1 always 1.5 The specified action is applied to all VMs in the vApp. All values other than 'default' ignore actions, order, and delay specified in the StartupSection. One of: powerOff (Power off the VMs. This is the default action if this attribute is missing or empty), suspend (Suspend the VMs), shutdown (Shut down the VMs), force (Attempt to power off the VMs. Failures in undeploying the VM or associated networks are ignored. All references to the vApp and its VMs are removed from the database), default (Use the actions, order, and delay specified in the StartupSection).
