AllEULAsAccepted

Element:
AllEULAsAccepted
Type:
boolean
Namespace:
http://www.w3.org/2001/XMLSchema
Description:
Used to confirm acceptance of all EULAs in a vApp template. Instantiation fails if this element is missing, empty, or set to false and one or more EulaSection elements are present.
Since:
0.9
XML Representation:
<AllEULAsAccepted> xs:boolean </AllEULAsAccepted>
